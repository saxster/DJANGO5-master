### **2. Multi-Framework Question System (5 Frameworks)**

#### **5 Whys** (Root Cause Analysis)
```
Q1: "Why did the pump fail?"
A1: "Bearing seized"

Q2: "Why did the bearing seize?"
A2: "No lubrication"

Q3: "Why was there no lubrication?"
A3: "Maintenance was skipped"

Q4: "Why was maintenance skipped?"
A4: "Parts were unavailable"

Q5: "Why were parts unavailable?"
A5: "Procurement delay due to budget freeze"

ROOT CAUSE: Budget freeze â†’ procurement delay â†’ maintenance skip â†’ failure
```

#### **SBAR** (Medical/Safety Framework)
```
Situation: What exactly happened? (facts only)
Background: What were conditions leading to this?
Assessment: What do you think caused it? What are risks?
Recommendation: What specific actions needed?
```

#### **5W1H** (Complete Detail Capture)
```
Who: was involved?
What: exactly happened?
When: did it occur?
Where: specifically?
Why: did it happen?
How: did events unfold?
```

#### **Ishikawa/Fishbone** (6M Causal Mapping)
```
People: Training, skills, staffing issues?
Process: Procedure, workflow, documentation issues?
Equipment: Tools, machinery, maintenance issues?
Environment: Workspace, weather, safety issues?
Materials: Supply, quality, inventory issues?
Management: Planning, oversight, policy issues?
```

#### **STAR** (Behavioral/Performance)
```
Situation: What was the context?
Task: What was supposed to happen?
Action: What actually occurred?
Result: What was the outcome?
```

**AI auto-selects framework based on report type and learns which works best.**
